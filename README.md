# Netty 기반 TCP 서버 학습용 프로젝트

이 프로젝트는 Java와 Netty를 사용하여 TCP 소켓 서버를 구축하는 방법을 학습하는 것을 목표로 합니다. Spring Boot 프레임워크 위에서 Netty 서버를 설정하고 운영하는 방법에 중점을 둡니다.

현재는 프로젝트의 뼈대를 기획 및 구축하고, 예시로 클라이언트의 접속 및 인증 처리를 담당하는 `Auth` 관련 기능의 기초만 구현되어 있는 초기 단계입니다.

## 🌟 주요 학습 포인트

- **Netty 서버 생명주기 관리**: Spring Boot와 통합하여 Netty 서버(`ServerBootstrap`)를 설정하고 관리합니다.
- **채널 파이프라인 구성**: 클라이언트가 접속할 때 생성되는 채널에 `Decoder`, `Encoder`, `Handler` 등을 파이프라인으로 구성하여 데이터를 처리하는 방법을 학습합니다.
- **커스텀 프로토콜 구현**: `AuthDecoder`를 통해 서버가 정의한 규칙에 따라 Byte Stream을 구조화된 메시지 객체로 변환하는 방법을 익힙니다.
- **외부 설정을 통한 서버 관리**: `application.yml` 파일에 서버의 포트, 스레드 개수 등의 설정을 분리하여 유연성을 높입니다.

## 🚀 실행 방법

1.  **프로젝트 클론 및 빌드**:
    ```bash
    git clone <repository-url>
    cd NettyPractice
    ./mvnw clean install
    ```

2.  **애플리케이션 실행**:
    Maven Spring Boot 플러그인을 사용하여 애플리케이션을 실행합니다.
    ```bash
    ./mvnw spring-boot:run
    ```
    서버는 `application.yml`에 설정된 `auth.host`와 `auth.port` (기본: 127.0.0.1:9960)에서 실행됩니다.
